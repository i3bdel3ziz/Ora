Timesheet App — V1.7.6

Lightweight Sun–Thu (Qatar) timesheet tracker.

Features:
- 8h daily target with one 9.5h "long day"
- Fill all / Clear all
- Weekly subtotals and averages
- Trend chart with 8h target
- Progress ring with completion %
- Quick month jump, reset, prev/next
- Export: CSV, PDF, JSON backup/restore
- Mobile sticky totals bar
- LocalStorage persistence
- Footer shows build number and timestamp
